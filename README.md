- Practice writing clean, readable code

- Identify common interview patterns

## Topics Covered
- Arrays & Strings
- Hash Maps
- Two Pointers
- Recursion
- Trees
- Basic Dynamic Programming

Each solution includes;
- Approach explanation
- Time & space complexity
